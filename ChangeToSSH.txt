First go to your repo where the project is.
run "git remote -v"
Go to bitbucket website and at the overview copy the SSH address. EXCLUDE "git
clone" just copy the ADDRESS!!! 
go back to the terminal and run "git remote set-url origin
git@github.com:ADDRESS" insert the url in ADDRESS you copied from the OVERVIEW. 

~~~~~if successful this is the output when running "git remote -v"~~~~~~~~~~~~~
origin  git@github.com:ADDRESS (fetch)
origin  git@github.com:ADDRESS (push)
